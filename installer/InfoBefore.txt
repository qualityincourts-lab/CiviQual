ABOUT THIS SOFTWARE
-------------------
CIVIQUAL (Workbench for Analysis, Testing, Statistics, Optimization &
Navigation) is statistical analysis software for Lean Six Sigma.
This software is NOT affiliated with IBM Corporation or IBM CiviQual.


WHAT'S NEW IN VERSION 1.3.0
---------------------------
This release focuses on Section 508 accessibility compliance:

• Enhanced focus indicators (gold outline on buttons)
• Background highlight on focused input fields
• Accessible names and descriptions on all controls
• Label associations for screen readers
• Tooltips on all input fields
• Help → Keyboard Shortcuts dialog
• Help → Accessibility Information dialog
• Improved disabled button contrast
• Full keyboard navigation support
• Compatible with NVDA, JAWS, and Windows Narrator


SYSTEM REQUIREMENTS
-------------------
• Windows 10 or later (64-bit)
• 4 GB RAM minimum (8 GB recommended)
• 200 MB available disk space
• 1280x800 minimum display resolution
