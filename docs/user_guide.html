Getting Started

CiviQual Stats is a statistical analysis application designed for Lean Six Sigma practitioners in government agencies, courts, and public-sector organizations.

Quick Start

  1. Load your data — Click Ctrl+O or use File → Open Data File
  2. Select a column — Use the Column dropdown in the toolbar
  3. Choose a tool — Navigate to the appropriate DMAIC phase tab
  4. Run analysis — Press F5 or click Run Analysis
  5. Export results — Use Ctrl+E to export charts

Loading Data

Supported Formats

  • CSV — Comma-separated values (.csv)
  • Excel — Microsoft Excel (.xlsx, .xls)

Data Requirements

  • First row should contain column headers
  • Numeric columns are automatically detected for analysis
  • Date columns should be in standard formats (YYYY-MM-DD, MM/DD/YYYY)

Tools Reference

Control Charts

Control charts detect process variation using statistical limits. CiviQual Stats supports:

  • I-Chart — Individual measurements
  • MR-Chart — Moving range
  • I-MR Chart — Combined view

Western Electric rules are automatically applied to detect out-of-control conditions:

  • Rule 1: One point beyond 3σ
  • Rule 2: Two of three points beyond 2σ (same side)
  • Rule 3: Four of five points beyond 1σ (same side)
  • Rule 4: Eight consecutive points on same side of center
  • Rule 5: Six consecutive points increasing or decreasing
  • Rule 6: Fourteen consecutive points alternating up and down

Capability Analysis

Process capability indices measure how well your process meets specifications:

  • Cp — Potential capability (spread vs. specification width)
  • Cpk — Actual capability (accounts for centering)
  • Pp — Overall performance (long-term)
  • Ppk — Overall performance with centering
A Cpk ≥ 1.33 is generally considered capable. For critical processes, Cpk ≥ 1.67 may be required.

ANOVA

Analysis of Variance tests whether group means are significantly different. Includes:

  • One-way ANOVA with F-statistic and p-value
  • Tukey HSD post-hoc test for pairwise comparisons
  • Effect size (eta-squared)
