Introduction

CiviQual Stats is a free, open-source statistical process control and Lean Six Sigma analytics application designed for public-sector organizations. It organizes its tools around the DMAIC improvement cycle, providing a structured workflow for quality management activities. A separately licensed Pro edition unlocks the Green Belt and Black Belt toolkit.

Getting Started

Loading data

From the File menu choose Open Data… and select a CSV, TSV, or Excel file. The first row is treated as column headers. After a file loads, a status line at the bottom shows the active dataset.

Navigating the DMAIC workspace

The main window uses top-level tabs for each DMAIC phase. Within each phase, left-side sub-tabs list the available tools. Free tools are always available; Pro tools appear only when a valid Pro license is installed.

Running analyses

Each tool panel follows the same pattern: inputs on the left, a chart in the upper right, and a results pane at the bottom. Click Run Analysis to execute the current tool.

Define Phase

Define tools help scope a project and document processes.

  • SIPOC — Suppliers, Inputs, Process, Outputs, Customers.
  • Process Map — High-level linear process diagram.
  • RACI — Responsibility assignment matrix.
  • Data Sampling — Random, systematic, or range-based sampling.
  • Split Worksheet — Export subsets by a grouping column.

Measure Phase

Measure tools establish baselines and evaluate current performance.

  • 4-Up Chart — Histogram, run chart, X-chart, and probability plot.
  • Descriptive Stats — Summary statistics and Anderson-Darling normality test.
  • Control Charts (XmR) — Individuals and moving range chart with Western Electric rule detection.
  • Capability — Cp, Cpk, Pp, Ppk and PPM projections.
  • Probability Plot — Normal probability plot.
  • Histogram — Frequency distribution with normal overlay.
  • Box Plot — Quartile distribution view.
  • Run Chart — Time series with runs test and Mann-Kendall trend.
  • Pro MSA (Gage R&R), Sample Size, Advanced Capability (Box-Cox).
Western Electric Rules detect: (1) points beyond 3σ, (2) 9 points on one side, (3) 6 trending, (4) 14 alternating, (5) 2 of 3 beyond 2σ, (6) 4 of 5 beyond 1σ, (7) 15 within 1σ (stratification), and (8) 8 beyond 1σ (mixture).

Analyze Phase

  • ANOVA — One-way ANOVA with Tukey HSD post-hoc comparisons.
  • Correlation — Pearson and Spearman correlation.
  • Pareto — Vital few analysis with cumulative percentage.
  • Swim Lane, VSM, Fishbone — Process and cause mapping diagrams.
  • Pro Hypothesis Tests, DOE (2^k factorial), Multiple Regression, Root Cause (5 Whys, Is/Is-Not), Data Tools (outlier detection and transforms).

Improve Phase

  • ROI Calculator — Cumulative net benefit projection.
  • Pro Solution Tools (PICK chart) and Lean Calculators (Takt, OEE, PCE, Little's Law).

Control Phase

  • Control Chart Review — Rule-based review of a recent window.
  • Pro CUSUM / EWMA, Planning Tools (FMEA and Control Plan), Chart Editor.

Pro License

A Pro license is a signed JSON file. To install:

  1. Open the License menu and choose Install License File….
  2. Select the license file provided by the publisher.
  3. Restart CiviQual Stats. Pro sub-tabs appear in each phase.

The application uses RSA-PSS signature verification to authenticate the license. If the signature is invalid, expired, or missing the cryptography package, the application runs in Free mode and displays a diagnostic in the License Status dialog.

Troubleshooting

Pro tools do not appear after installing a license

Restart the application. The license file is read only at startup.

The application fails to launch

Open a command prompt, activate the Python environment, and run python main.py to see the error trace. Confirm that every dependency in requirements.txt is installed.

Plots look small or misaligned on a high-DPI display

Launch from the MSI installer rather than a manually unpacked copy; the installer sets the high-DPI environment variable. When running from source, ensure QT_AUTO_SCREEN_SCALE_FACTOR=1 is in the environment.
