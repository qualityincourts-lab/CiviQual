\## CiviQual Stats v1.2.0

CiviQual is now the master brand for a suite of public-sector quality management tools. This application has been renamed from \*\*CiviQual\*\* to \*\*CiviQual Stats\*\* to reflect its focus on statistical process control.



\*\*CiviQual Suite Family:\*\*

\- \*\*CiviQual Stats\*\* — Statistical process control (this product)

\- \*\*CiviQual Tier\*\* — G1:2026 maturity assessment (in development)

\- \*\*CiviQual Risk\*\* — Enterprise risk management platform (planned)

\### Installation



\*\*Windows Installer (Recommended):\*\*

Download `CiviQualStats\_1.2.0.msi` and run. Signed with Certum code signing certificate.



\*\*From Source:\*\*

1\. Clone the repository

2\. Run `pip install -r requirements.txt`

3\. Launch with `python main.py`



\### Upgrade Notes



\- Config directory changed from `CiviQual` to `CiviQualStats`

\- Existing v1.1.0 installations will continue to work

\- New installations use CiviQual Stats branding
